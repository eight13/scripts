## 文档规范

### 存放位置

- 项目级文档：`doc/README.md`（本文件）
- 模块级文档：`<模块>/doc/` 目录下
- 跨模块文档：`doc/` 根目录

### 写作风格

- 面向不熟悉架构的读者
- 先现象后原理，用类比解释
- 日常使用命令放最前面，技术细节放后面

### 新建文档流程

1. 在对应位置建 `doc/` 目录
2. 创建 `README.md` 索引
3. 更新本文件（总索引）
4. 小修复合并到已有文档，不单独开文件
